# 🚀 Advanced Recon & Automation Framework
An automated Python-based cybersecurity reconnaissance tool designed for fast and smart network scanning, directory bruteforcing, and subdomain enumeration.

## ✨ Features
* **Smart Port Scanner & Service Detector:** Fast multi-threaded port discovery.
* **Directory Bruteforcer:** Finds hidden files and sensitive endpoints on web servers.
* **Subdomain Enumerator (DNS Recon):** Discovers active subdomains and automatically saves results into a structured `.txt` report file.
